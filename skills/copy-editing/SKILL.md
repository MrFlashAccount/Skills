Use this skill for existing market-facing copy.

The Marketing role supplies the judgment. This skill supplies the editing-pass workflow, refresh decisions, and delivery format.

## Routing boundary

Use this skill when copy already exists and the main job is improving it without silently changing the core strategic goal.

Use it for:
- tightening or polishing page copy
- proof, objection, or CTA improvements
- outdated-page refreshes
- copy sweeps across a landing page, pricing page, email, sales one-pager, or campaign asset

Do not use it for:
- new page creation or major repositioning; use `copywriting`
- repository README openings, developer-facing product framing, or technical feature blurbs; use `devrel-copywriter`
- setup, onboarding, tutorial, or API docs; use `docs-writer`

## Task class

- `polish`: localized edits where the audience, angle, and structure remain intact.
- `sweep`: multi-pass edit where the message mostly stays but clarity, proof, specificity, and friction need work.
- `refresh`: published or older copy that may need factual, proof, or positioning updates.

If the current copy is strategically wrong for the audience or surface, escalate to `copywriting` instead of hiding a rewrite inside editing.

## Workflow

- `polish`: contract -> edit -> one review.
- `sweep`: contract -> baseline read -> editing passes -> review checkpoint -> fix -> final packaging.
- `refresh`: contract -> refresh-vs-rewrite decision -> editing passes -> factual accuracy check -> final packaging.

## Rules

- Preserve the real strategic goal unless the user asks for a different angle.
- Load shared GTM context before judging audience fit or brand-language drift.
- Separate market-facing copy from DevRel/docs work.
- Do not upgrade weak proof into strong claims.
- Flag unsupported metrics, testimonials, customer language, and competitive assertions.
- If a claim cannot be proved, soften it or mark it as a proof gap.
- Keep edits traceable: the user should be able to see what changed and why.
- Near CTAs, prioritize friction removal, next-step clarity, and honest objection handling.
