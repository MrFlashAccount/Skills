# Fresh Evidence Source Guide

Use this only when existing evidence is not enough.

## Fast source selection

| Need | Best starting sources |
| --- | --- |
| category pains and alternatives | G2, Capterra, TrustRadius, competitor reviews |
| raw customer language | Reddit, niche forums, YouTube comments, community threads |
| trigger events and buying context | sales calls, churn notes, LinkedIn posts, job listings |
| objection patterns | demos, lost deals, review complaints, support tickets |
| proof expectations | case studies, review praise, switcher stories, implementation concerns |
| where the ICP hangs out | communities, newsletters, podcasts, creators, partner ecosystems |

## Extraction discipline

Capture:
- exact quote when possible
- source and date
- segment clues
- what prompted the statement
- whether it is pain, trigger, objection, alternative, or desired outcome

Do not paraphrase away useful wording.
Do not treat anonymous internet claims as universal truth.
Do not combine unlike roles into one segment just because they share a category.
