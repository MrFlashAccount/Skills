# Output Formats

Default package:

## Copy
- structured by surface or section
- headline / subheadline / CTA called out explicitly when relevant
- optional variants only for high-leverage lines such as headline, CTA, or hook

## Notes
- audience and channel assumptions
- proof used
- proof missing or softened claims
- why the main angle was chosen

## When useful
- 2-3 headline or CTA alternatives with one-line rationale each
- meta title / meta description for page work
- short test matrix for variants when the user wants experimentation

Do not bury the final copy inside long explanation.
