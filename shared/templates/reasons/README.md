# REASONS Template

Structured prompt/spec template for immutable REASONS Canvas artifacts.

Use [`reasons-canvas-template.md`](reasons-canvas-template.md) for any REASONS Canvas artifact. Artifact identity, lifecycle, lineage, approval, and downstream routing live outside this template.

The template inlines the REASONS Canvas construction guidance needed by agents in this repo. It defines artifact content only. It must not encode lifecycle state, approval gates, or process routing.
